# ============================================================
# 免疫相关关键词白名单
# 用于 PubMed 文献筛选模式（IMMUNE_FILTER_MODE=True）
# ============================================================
#
# 格式规则（v2 — 支持别名映射）：
#
#   ## 分组名             → 分组标题
#   标准名称              → 单一名称，无别名（旧格式，向后兼容）
#   标准名称 | 别名1 | 别名2 | ...
#                         → | 左侧第一个为「标准名称」（用于 downloaded_index/ 目录名）
#                           | 右侧为该指标的其他别名（用于 PubMed 搜索）
#                           → 所有别名命中均保存到「标准名称」目录下
#   # 注释               → 单行注释，忽略
#   空行                  → 忽略
#
# PubMed 查询说明：
#   标准名称和所有别名都会参与 PubMed 查询（[tiab] 字段）
#   命中任意一个别名 → 文章保存到「标准名称」对应的目录
#
# 示例：
#   TNF-alpha | TNF | tumor necrosis factor alpha | TNF-α
#   → 搜索 TNF-alpha[tiab] OR TNF[tiab] OR "tumor necrosis factor alpha"[tiab] OR TNF-α[tiab]
#   → 命中后统一保存到 downloaded_index/TNF-alpha/
#
# ============================================================
